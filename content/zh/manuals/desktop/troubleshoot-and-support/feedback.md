有多种方式可以对 Docker Desktop 或其功能提供反馈。

### 产品内反馈

在 Docker Desktop 控制面板的每个视图中，都有一个 **Give feedback** 链接。这将打开一个反馈表单，您可以直接向 Docker 团队分享您的想法。

### 通过 Docker 社区论坛反馈

要获取社区帮助、查看当前用户话题、加入或发起讨论，请登录相应的 Docker 论坛：

- [Docker Desktop for Mac 论坛](https://forums.docker.com/c/docker-for-mac)
- [Docker Desktop for Windows 论坛](https://forums.docker.com/c/docker-for-windows)
- [Docker Desktop for Linux 论坛](https://forums.docker.com/c/docker-desktop-for-linux/60)

### 在 GitHub 上报告 Bug 或问题

要报告 Bug 或问题，请访问：
- [GitHub 上的 Docker Desktop for Mac 问题列表](https://github.com/docker/for-mac/issues)
- [GitHub 上的 Docker Desktop for Windows 问题列表](https://github.com/docker/for-win/issues)
- [GitHub 上的 Docker Desktop for Linux 问题列表](https://github.com/docker/desktop-linux/issues)
- [GitHub 上的 Docker Extensions 问题列表](https://github.com/docker/extensions-sdk/issues)

### 通过社区 Slack 频道反馈

您还可以通过以下 [Docker 社区 Slack](https://dockr.ly/comm-slack) 频道提供反馈：

- #docker-desktop-mac
- #docker-desktop-windows
- #docker-desktop-linux
- #extensions
