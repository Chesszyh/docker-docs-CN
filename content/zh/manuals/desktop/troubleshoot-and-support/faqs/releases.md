### 新版本发布的频率是多少？

新版本大约每月发布一次，除非有需要尽快发布的关键修复程序。

**Software updates** 选项卡中的 **Automatically check for updates** 设置默认处于开启状态。这意味着当有新版本可用时，您会在 Docker 菜单中收到通知，并且 Docker Desktop 控制面板上会显示通知徽章。

您还可以通过勾选 **Always download updates** 复选框，让 Docker Desktop 在后台自动下载新更新。

有时，新版本会在几天内逐步推出。因此，如果您稍微等待一下，它很快就会出现。或者，您可以选择 Docker 菜单中的 **Check for updates** 来立即获取最新版本。

### 如何确保我组织中的所有用户都使用相同的版本？

这需要通过您 IT 管理员的终端管理 (MDM) 软件进行统一管理。
