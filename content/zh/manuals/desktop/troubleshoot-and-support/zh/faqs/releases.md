### 新版本的发布频率是多少？

除非有需要尽快发布的关键修复，否则新版本大约每月发布一次。

**Software updates** 选项卡中的 **Automatically check for updates** 设置默认为开启。这意味着当有新版本可用时，您会在 Docker 菜单中收到通知，并在 Docker Desktop Dashboard 上看到通知徽章。

您还可以通过选中 **Always download updates** 复选框，让 Docker Desktop 在后台自动下载新更新。

有时新版本会在几天内逐步推出。因此，如果您等待一下，它很快就会出现。或者，您可以在 Docker 菜单中选择 **Check for updates** 以立即获取最新版本。

### 如何确保组织中的所有用户使用相同的版本？

这通过您的 IT 管理员的端点管理软件来管理。
