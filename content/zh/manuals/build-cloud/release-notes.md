本页面包含 Docker Build Cloud 发行版中的新特性、改进、已知问题和 Bug 修复信息。

## 2025-02-24

### 新增

新增了 **Build settings**（构建设置）页面，您可以在其中为组织中的云构建器配置磁盘分配、私有资源访问和防火墙设置。这些配置有助于优化存储、实现对私有镜像库的访问并保障出站网络流量的安全。
