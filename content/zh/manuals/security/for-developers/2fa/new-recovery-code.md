如果您丢失了双因素认证恢复码，但仍然可以访问您的 Docker Hub 账户，您可以生成新的恢复码。

1. 登录您的 [Docker 账户](https://app.docker.com/login)。
2. 选择您的头像，然后从下拉菜单中选择 **Account settings**。
3. 选择 **2FA**。
4. 输入您的密码，然后选择 **Confirm**。
5. 选择 **Generate new code**。

这将生成一个新的恢复码。选择可见性图标以查看该验证码。请记住保存您的恢复码并将其存放在安全的地方。
