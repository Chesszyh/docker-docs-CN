如果您丢失了双重身份验证恢复码，但仍然可以访问您的 Docker Hub 帐户，则可以生成一个新的恢复码。

1. 登录您的 [Docker 帐户](https://app.docker.com/login)。
2. 选择您的头像，然后从下拉菜单中选择 **Account settings（帐户设置）**。
3. 选择 **2FA**。
4. 输入您的密码，然后选择 **Confirm（确认）**。
5. 选择 **Generate new code（生成新代码）**。

这将生成一个新的恢复码。选择可见性图标（眼睛图标）以查看代码。请务必保存您的恢复码并将其存放在安全的地方。
