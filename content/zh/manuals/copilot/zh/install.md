要使用 Docker for GitHub Copilot 扩展，您首先需要为您的组织[安装](#install)扩展，并[管理](#manage-policies) Copilot 在您组织中的策略。

## 前提条件

在开始之前，请确保您已在 GitHub.com 上登录您的 GitHub 账户。

## 安装

要为您的 GitHub 组织安装 Docker for GitHub Copilot 扩展：

1. 前往 GitHub Marketplace 中的 [Docker for GitHub Copilot](https://github.com/marketplace/docker-for-github-copilot) 应用。

2. 选择页面顶部的 **Add** 按钮。

3. 在 **Pricing and setup** 下，选择您想要安装扩展的组织，然后选择 **Install it for free**。

4. 选择 **Complete order and begin installation** 按钮。

5. 选择您想要使用 Docker Extension for GitHub Copilot 的仓库，然后点击 **Install** 完成安装。

## 管理策略

如果您要为 GitHub 组织启用此扩展，您还需要启用 Copilot Extensions 策略。有关说明，请参阅[在您的组织中为 GitHub Copilot Extensions 设置策略](https://docs.github.com/en/copilot/managing-copilot/managing-github-copilot-in-your-organization/setting-policies-for-copilot-in-your-organization/managing-policies-for-copilot-in-your-organization#setting-a-policy-for-github-copilot-extensions-in-your-organization)。
