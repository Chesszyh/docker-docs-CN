您可以更新个人账户或组织的账单信息。当您更新账单信息时，这些更改将应用于未来的账单发票。请注意，您无法更新现有的发票，包括已付和未付的发票。

提供的账单信息将出现在您所有的账单发票上。提供的电子邮件地址是 Docker 发送所有发票及其他 [账单相关通信](#更新您的账单发票电子邮件地址) 的地方。

## 管理账单信息

### 个人账户

{{< tabs >}}
{{< tab name="Docker 订阅" >}}

更新账单信息的步骤：

1. 登录 [Docker Home](https://app.docker.com/) 并选择您的组织。
2. 选择 **Billing**（账单）。
3. 从左侧导航中选择 **Billing information**（账单信息）。
4. 在您的账单信息卡片上，选择 **Change**（更改）。
5. 更新您的账单联系人和账单地址信息。
6. 可选：要添加或更新增值税 (VAT) ID，请勾选 **I'm purchasing as a business**（我正在作为企业购买）复选框并输入您的纳税人识别号 (Tax ID)。

    > [!IMPORTANT]
    >
    > 您的增值税 (VAT) 号码必须包含国家前缀。例如，如果您输入德国的 VAT 号码，应输入 `DE123456789`。

7. 选择 **Update**（更新）。

{{< /tab >}}
{{< tab name="旧版 Docker 订阅" >}}

更新账单信息的步骤：

1. 登录 [Docker Hub](https://hub.docker.com)。
2. 选择您的组织，然后选择 **Billing**（账单）。
3. 选择 **Billing Address**（账单地址）并输入您更新后的账单信息。
4. 可选：要添加或更新增值税 (VAT) ID，请输入您的 **Tax ID/VAT**。

    > [!IMPORTANT]
    >
    > 您的增值税 (VAT) 号码必须包含国家前缀。例如，如果您输入德国的 VAT 号码，应输入 `DE123456789`。

5. 选择 **Submit**（提交）。

{{< /tab >}}
{{< /tabs >}}

## 更新您的账单发票电子邮件地址

Docker 会发送以下与账单相关的电子邮件：

- 新订阅确认。
- 已付发票确认。
- 信用卡或借记卡付款失败通知。
- 信用卡或借记卡过期通知。
- 订阅取消确认。
- 为按年订阅用户发送的订阅续订提醒。该提醒会在续订日期前 14 天发送。

您可以随时更新接收账单发票的电子邮件地址。

### 个人账户

{{< tabs >}}
{{< tab name="Docker 订阅" >}}

更新账单电子邮件地址的步骤：

1. 登录 [Docker Home](https://app.docker.com/) 并选择您的组织。
2. 选择 **Billing**（账单）。
3. 从左侧导航中选择 **Billing information**（账单信息）。
4. 在您的账单信息卡片上，选择 **Change**（更改）。
5. 更新您的账单联系信息并选择 **Update**（更新）。

{{< /tab >}}
{{< tab name="旧版 Docker 订阅" >}}

更新账单电子邮件地址的步骤：

1. 登录 [Docker Hub](https://hub.docker.com)。
2. 选择您的组织，然后选择 **Billing**（账单）。
3. 选择 **Billing Address**（账单地址）。
4. 在 **Billing contact**（账单联系人）部分更新电子邮件地址。
5. 选择 **Submit**（提交）。

{{< /tab >}}
{{< /tabs >}}
