如果您是美国客户且享有销售税豁免，您可以向 Docker 支持团队注册有效的税务豁免证明。如果您是需要缴纳 VAT（增值税）的全球客户，请确保提供您的 [VAT 号码](/billing/history/#include-your-vat-number-on-your-invoice)，包括 VAT 国家前缀。

{{% include "tax-compliance.md" %}}

## 前提条件

在提交税务豁免证明之前，请确保以下几点。

1. 您的客户名称与豁免证明上的名称相匹配
2. 您的税务豁免证明将 Docker Inc 指定为卖方或供应商，并填写了所有适用信息
3. 您的证明已签名并注明日期，且到期日期尚未过期
4. 您拥有要应用税务豁免证明的账户的有效 Docker ID/命名空间

## 注册税务证明

1. [提交 Docker 支持工单](https://hub.docker.com/support/contact?topic=Billing&subtopic=Tax%20information)以启动注册税务证明的流程。
2. 输入所需信息。
3. 在 **Additional Information** 字段中，列出您要应用税务豁免证明的账户的 Docker ID/命名空间。

   > [!TIP]
   >
   > 如果适用，您可以列出共享同一税务豁免证明的多个命名空间。
4. 通过将税务证明拖放到文件区域来从您的系统添加税务证明，或选择 **Browse Files** 按钮打开文件对话框。
5. 选择 **Submit**。

如果需要任何其他信息，Docker 支持团队将与您联系。当您的税务豁免状态应用到您的账户后，您将收到来自 Docker 的电子邮件确认。
