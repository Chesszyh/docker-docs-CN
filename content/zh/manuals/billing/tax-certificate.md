如果您是美国客户且免征销售税，您可以在 Docker 支持团队登记有效的免税证明。如果您是受增值税 (VAT) 约束的全球客户，请确保提供您的 [VAT 号码](/billing/history/#在发票中包含您的-vat-号码)，包括您的 VAT 国家前缀。

{{% include "tax-compliance.md" %}}

## 前提条件

在提交免税证明之前，请确保以下几点：

1. 您的客户名称与免税证明上的名称一致
2. 您的免税证明将 Docker Inc 指定为卖家（Seller）或供应商（Vendor），并且填写了所有适用信息
3. 您的证明已签字并注明日期，且未超过有效期
4. 您拥有要应用免税证明的账户的有效 Docker ID/命名空间

## 登记免税证明

1. [提交 Docker 支持工单](https://hub.docker.com/support/contact?topic=Billing&subtopic=Tax%20information) 以启动免税证明登记流程。
2. 输入所需信息。
3. 在 **Additional Information**（额外信息）字段中，列出您希望应用该免税证明的账户的 Docker ID/命名空间。

   > [!TIP]
   >
   > 如果适用，您可以列出共享同一份免税证明的多个命名空间。
4. 通过拖放将系统中的税务证明添加到文件区域，或者选择 **Browse Files**（浏览文件）按钮打开文件对话框。
5. 选择 **Submit**（提交）。

如果需要任何额外信息，Docker 支持团队将与您联系。一旦免税状态应用到您的账户，您将收到 Docker 的电子邮件确认。
