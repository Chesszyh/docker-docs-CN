## 17.11.0-ce
2017-11-20

> [!IMPORTANT]
> Docker CE 17.11 是第一个基于 [containerd 1.0 beta](https://github.com/containerd/containerd/releases/tag/v1.0.0-beta.2) 的 Docker 版本。Docker CE 17.11 及更高版本不能识别使用之前 Docker 版本启动的容器。如果您使用 Live Restore，您必须在升级到 Docker CE 17.11 之前停止所有容器。否则，任何由 17.11 之前的 Docker 版本启动的容器在升级后将不会被 Docker 识别，并将继续在系统上以非托管方式运行。
