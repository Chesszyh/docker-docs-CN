Docker Desktop 根据 Docker 订阅服务协议（Docker Subscription Service Agreement）获得许可。当您下载并安装 Docker Desktop 时，系统会要求您同意更新后的条款。

我们的 [Docker 订阅服务协议](https://www.docker.com/legal/docker-subscription-service-agreement)规定：

 - Docker Desktop 对小型企业（员工少于 250 人且年收入少于 1000 万美元）、个人使用、教育和非商业开源项目免费。
 - 否则，专业使用需要付费订阅。
- 政府实体也需要付费订阅。
- Docker Pro、Team 和 Business 订阅包括 Docker Desktop 的商业使用。

阅读[博客](https://www.docker.com/blog/updating-product-subscriptions/)和 [Docker 订阅常见问题](https://www.docker.com/pricing/faq)以了解这可能如何影响使用 Docker Desktop 的公司。

> [!NOTE]
>
> Docker 和 Moby 开源项目（如 Docker Engine）的许可和分发条款不会改变。

Docker Desktop 使用开源软件构建。有关 Docker Desktop 中开源组件许可的信息，请选择鲸鱼菜单 > **About Docker Desktop** > **Acknowledgements**。

Docker Desktop 分发了一些根据 GNU 通用公共许可证授权的组件。点击[此处](https://download.docker.com/opensource/License.tar.gz)下载这些组件的源代码。

> [!TIP]
>
> 探索 [Docker 订阅](https://www.docker.com/pricing/)以了解 Docker 还能为您提供什么。
