Docker 强化镜像（Docker Hardened Images，DHIs）是由 Docker 维护的精简、安全、生产就绪的容器基础镜像和应用镜像。DHIs 旨在减少漏洞并简化合规性，可轻松集成到您现有的基于 Docker 的工作流程中，几乎无需重新配置工具。

浏览以下章节，开始使用 Docker 强化镜像，将其集成到您的工作流程中，并了解是什么使它们安全且适合企业使用。
