Docker 扩展（Docker Extensions）让您能够在 Docker Desktop 中使用第三方工具来扩展其功能。

您可以将常用的开发工具无缝连接到应用程序开发和部署工作流程中。通过调试、测试、安全和网络功能来增强 Docker Desktop，并使用扩展 [SDK](extensions-sdk/_index.md) 创建自定义插件。

任何人都可以使用 Docker 扩展，并且对可安装的扩展数量没有限制。

![Extensions Marketplace](/assets/images/extensions.webp)

## 有哪些可用的扩展？

目前有合作伙伴和社区构建的扩展，以及 Docker 官方构建的扩展。
您可以在 [Docker Hub](https://hub.docker.com/search?q=&type=extension) 或 Docker Desktop 内的扩展市场（Extensions Marketplace）中浏览可用扩展列表。
