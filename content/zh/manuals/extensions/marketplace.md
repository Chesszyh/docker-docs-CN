扩展市场（Extensions Marketplace）中有两种类型的扩展：
- Docker 审核的扩展
- 自行发布的扩展

Docker 审核的扩展由 Docker 扩展团队手动审核，以确保额外的信任度和质量水平。它们在市场中显示为 **Reviewed**（已审核）。

自行发布的扩展由扩展开发者自主发布，并经过自动化验证流程。它们在市场中显示为 **Not reviewed**（未审核）。

## 安装扩展

> [!NOTE]
>
> 对于某些扩展，需要在使用前单独创建账户。

安装扩展的步骤：

1. 打开 Docker Desktop。
2. 在 Docker Desktop 仪表板中，选择 **Extensions** 选项卡。
   扩展市场会在 **Browse** 选项卡中打开。
3. 浏览可用的扩展。
   您可以按 **Recently added**（最近添加）、**Most installed**（安装最多）或按字母顺序对扩展列表进行排序。或者，使用 **Content** 或 **Categories** 下拉菜单按是否已审核或按类别搜索扩展。
4. 选择一个扩展并点击 **Install**。

完成后，您可以选择 **Open** 来访问该扩展或安装其他扩展。该扩展还会出现在左侧菜单和 **Manage** 选项卡中。

## 更新扩展

您可以在 Docker Desktop 发布版本之外更新任何扩展。要将扩展更新到最新版本，请导航到 Docker Desktop 仪表板并选择 **Manage** 选项卡。

**Manage** 选项卡会显示所有已安装的扩展。如果某个扩展有新版本可用，它会显示一个 **Update** 按钮。


## 卸载扩展

您可以随时卸载扩展。

> [!NOTE]
>
> 扩展使用的任何存储在卷中的数据必须手动删除。

1. 导航到 Docker Desktop 仪表板并选择 **Manage** 选项卡。
   这会显示您已安装的扩展列表。
2. 选择要卸载的扩展右侧的省略号。
3. 选择 **Uninstall**。
