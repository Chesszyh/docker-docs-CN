有多种方式可以为 Docker Compose 提供反馈。

### 在 GitHub 上报告 Bug 或问题

要报告 Bug 或问题，请访问 [GitHub 上的 Docker Compose](https://github.com/docker/compose/issues)。

### 通过社区 Slack 频道提供反馈

您也可以通过 [Docker 社区 Slack](https://dockr.ly/comm-slack) 的 `#docker-compose` 频道提供反馈。
