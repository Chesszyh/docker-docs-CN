以下示例展示了如何使用 Docker Compose 的各个方面。作为前提条件，如果您尚未安装，请务必先 [安装 Docker Compose](/manuals/compose/install/_index.md)。

## 这些示例涵盖的关键概念

这些示例应该能帮助您：

- 使用 [Compose 文件](/reference/compose-file/_index.md) 定义基于 Docker 镜像的服务
- 理解 `compose.yaml` 与 [Dockerfile](/reference/dockerfile/) 之间的关系
- 了解如何从 Compose 文件中调用您的应用程序服务
- 了解如何将应用程序和服务部署到 [swarm](/manuals/engine/swarm/_index.md)

## Awesome Compose 示例

Awesome Compose 示例提供了一个起点，介绍如何使用 Docker Compose 集成不同的框架和技术。所有示例都可在 [Awesome-compose GitHub 仓库](https://github.com/docker/awesome-compose) 中找到，并且可以直接通过 `docker compose up` 运行。
