> **Docker Debug 介绍**
>
> 要轻松获取任何容器的调试 shell，请使用 `docker debug`。Docker Debug
> 是 `docker exec` 调试功能的替代方案。使用它，您可以进入任何容器或镜像的
> shell，即使是精简版镜像也可以，无需任何修改。此外，您还可以在其可定制的
> 工具箱中携带您喜爱的调试工具。
>
> 立即探索 [Docker Debug](../debug.md)。
