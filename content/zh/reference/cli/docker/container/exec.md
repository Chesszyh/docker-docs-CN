> **介绍 Docker Debug**
>
> 要轻松获取任何容器的调试 shell，请使用 `docker debug`。Docker Debug 是 `docker exec` 调试的替代品。使用它，您可以进入任何容器或镜像的 shell，即使是精简的容器或镜像，也无需修改。此外，您可以在其可自定义的工具箱中携带您喜欢的调试工具。
>
> 立即探索 [Docker Debug](../debug.md)。
