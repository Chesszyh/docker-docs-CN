Docker Scout 的[修复功能](/manuals/scout/policy/remediation.md)通过基于策略评估提供量身定制的建议，帮助您解决供应链和安全问题。这些建议指导您改善策略合规性或增强镜像元数据，使 Docker Scout 能够在未来执行更准确的评估。

您可以使用此功能确保基础镜像是最新的，并且供应链证明是完整的。当发生违规时，Docker Scout 会提供建议的修复方案，例如更新基础镜像或添加缺失的证明。如果没有足够的信息来确定合规性，Docker Scout 会建议采取措施来帮助解决问题。

在 Docker Scout 仪表板中，您可以通过查看违规或合规不确定性来查看和处理这些建议。通过与 GitHub 等工具的集成，您甚至可以自动化更新，直接从仪表板修复问题。
