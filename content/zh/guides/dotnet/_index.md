本 .NET 入门指南将教你如何使用 Docker 创建容器化的 .NET 应用程序。在本指南中，你将学习如何：

- 容器化并运行 .NET 应用程序
- 搭建本地开发环境，使用容器开发 .NET 应用程序
- 使用容器运行 .NET 应用程序的测试
- 使用 GitHub Actions 为容器化的 .NET 应用程序配置 CI/CD 流水线
- 将容器化应用程序本地部署到 Kubernetes，以测试和调试你的部署

完成 .NET 入门模块后，你应该能够根据本指南中提供的示例和说明，容器化你自己的 .NET 应用程序。

首先，让我们开始容器化一个现有的 .NET 应用程序。
