Java 入门指南教你如何使用 Docker 创建容器化的 Spring Boot 应用程序。在本模块中，你将学习如何：

- 容器化并运行使用 Maven 构建的 Spring Boot 应用程序
- 设置本地开发环境以将数据库连接到容器，配置调试器，并使用 Compose Watch 进行实时重载
- 在容器内运行单元测试
- 使用 GitHub Actions 为你的应用程序配置 CI/CD 管道
- 将容器化应用程序本地部署到 Kubernetes 以测试和调试部署

完成 Java 入门模块后，你应该能够根据本指南中提供的示例和说明容器化你自己的 Java 应用程序。
