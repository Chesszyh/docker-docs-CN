---
title: 为什么选择 Docker Build Cloud？
description: 了解 Docker Build Cloud 如何加速您的构建。
weight: 10
---

Docker Build Cloud 是一项服务，让您可以更快地构建容器镜像，
无论是在本地还是在 CI 中。构建在针对您的工作负载进行优化配置的云端基础设施上运行，
无需任何配置。该服务使用远程构建缓存，确保在任何地方以及所有团队成员都能快速构建。

Docker Build Cloud 相比本地构建提供了以下优势：

- 提升构建速度
- 共享构建缓存
- 原生多平台构建

您无需担心管理构建器或基础设施——只需连接到您的构建器并开始构建。分配给组织的每个云构建器都完全隔离在单独的 Amazon EC2 实例上，配备专用的 EBS 卷用于构建缓存，并在传输过程中进行加密。这意味着云构建器之间不存在共享的进程或数据。
