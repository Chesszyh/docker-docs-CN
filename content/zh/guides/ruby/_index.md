Ruby 语言特定指南教你如何使用 Docker 容器化 Ruby on Rails 应用程序。在本指南中，你将学习如何：

- 容器化并运行 Ruby on Rails 应用程序
- 配置 GitHub Actions 工作流以构建 Docker 镜像并将其推送到 Docker Hub
- 设置本地环境以使用容器开发 Ruby on Rails 应用程序
- 将容器化的 Ruby on Rails 应用程序本地部署到 Kubernetes 以测试和调试你的部署

从容器化现有的 Ruby on Rails 应用程序开始。
