PHP 语言特定指南教你如何使用 Docker 创建容器化的 PHP 应用程序。在本指南中，你将学习如何：

- 容器化并运行 PHP 应用程序
- 设置本地环境以使用容器开发 PHP 应用程序
- 在容器内运行 PHP 应用程序的测试
- 使用 GitHub Actions 为容器化 PHP 应用程序配置 CI/CD 管道
- 将容器化应用程序本地部署到 Kubernetes 以测试和调试部署

完成 PHP 语言特定指南后，你应该能够根据本指南中提供的示例和说明容器化你自己的 PHP 应用程序。
