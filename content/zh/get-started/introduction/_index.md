## 关于本系列

在本指南系列中，您将从在本地机器上安装和设置 Docker Desktop 开始，获得使用 Docker 的实践经验。您将学习如何运行您的第一个容器，了解容器化的基础知识及其好处。本系列指导您构建第一个 Docker 镜像，深入了解如何创建高效且可重用的镜像。最后，您将探索如何在 Docker Hub 上发布您的镜像，使您能够与更广泛的社区分享您的工作，并利用 Docker 强大的生态系统进行协作开发和部署。

## 您将学到什么

- 设置 Docker Desktop
- 运行您的第一个容器
- 构建您的第一个镜像
- 在 Docker Hub 上发布您的镜像
