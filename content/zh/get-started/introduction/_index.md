## 关于本系列

在本指南系列中，您将获得 Docker 的动手经验，从
在本地计算机上安装和设置 Docker Desktop 开始。您将
学习如何运行您的第一个容器，了解
容器化的基础知识及其好处。本系列指导您构建您的
第一个 Docker 镜像，提供有关创建高效和可重用
镜像的见解。最后，您将探索如何在 Docker Hub 上发布您的镜像，
使您能够与更广泛的社区分享您的工作，并利用
Docker 强大的生态系统进行协作开发和部署。
