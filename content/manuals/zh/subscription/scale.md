> [!NOTE]
>
> 旧版 Docker 订阅的所有者必须将其订阅升级为新的 Docker 订阅才能扩展其订阅。
>
> 旧版 Docker 订阅适用于最后一次购买或续订订阅在 2024 年 12 月 10 日之前的 Docker 订阅者。这些订阅者将保持其当前订阅和定价，直到其下一个续订日期在 2024 年 12 月 10 日或之后。要查看购买或续订历史记录，请查看您的[账单历史记录](../billing/history.md)。有关旧版订阅的更多详情，请参阅[宣布升级 Docker 计划](https://www.docker.com/blog/november-2024-updated-plans-announcement/)。

Docker 订阅让您可以随着需求的变化扩展您的消费。除了旧版 Docker 订阅外，所有付费 Docker 订阅都包含对 Docker Hub、Docker Build Cloud 和 Testcontainers Cloud 的访问权限以及基础消费量。请参阅 [Docker 订阅和功能](./details.md)以了解每个订阅附带多少基础消费量。您可以在订阅期间随时扩展您的消费。

您可以扩展以下消费：

- Docker Build Cloud 构建分钟数
- Docker Testcontainers Cloud 运行时分钟数

为了更好地了解您的需求，您可以随时查看您的消费情况。有关更多详情，请参阅[查看 Docker 产品使用情况](../admin/organization/manage-products.md#view-docker-product-usage)。

> [!WARNING]
>
> 订阅中包含的 Docker Build Cloud 和 Testcontainers 分钟数不会累积。额外分钟数在您的订阅期（月度或年度）结束时过期。例如，如果您有年度 Docker Team 订阅，包含 500 分钟，并购买了 500 额外分钟，只有 500 额外分钟会累积到您年度订阅期结束。

## 添加 Docker Build Cloud 构建分钟数

您可以在 Docker Build Cloud 仪表板中预购 Docker Build Cloud 构建分钟数：

1. 登录 [Docker Home](https://app.docker.com/) 并选择您的组织。
1. 选择 **Build Cloud**，然后选择 **Build minutes**。
1. 选择 **Add prepaid minutes**。
1. 选择您的额外分钟数量，然后选择 **Continue to payment**。
1. 输入您的付款详情和账单地址。
1. 查看您的订单并选择 **Pay**。

您的额外分钟数现在将显示在 Build minutes 页面上。

## 添加 Docker Testcontainers Cloud 运行时分钟数

您可以通过[联系销售](https://www.docker.com/pricing/contact-sales/)预购 Testcontainers Cloud 运行时分钟数。除了预购外，您还可以根据需要按需使用任意数量的分钟数。使用量将在每个月度计费周期结束时计费。按需使用的费率高于预购容量。为避免按需收费，请预购额外分钟数。
