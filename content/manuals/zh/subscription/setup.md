Docker 订阅提供功能和权益，以支持新手和专业开发者，以及为个人、团队和企业提供计划。要了解每个层级包含的内容，请参阅 [Docker 订阅和功能](./details.md)和 [Docker 定价](https://www.docker.com/pricing/)。

在本节中，了解如何开始为个人或组织设置 Docker 订阅。在开始之前，请确保您有一个 [Docker ID](../accounts/create-account.md)。

{{% include "tax-compliance.md" %}}

## 为个人账户设置 Docker 订阅

在您[创建 Docker ID](../accounts/create-account.md) 后，您将拥有一个 Docker Personal 订阅。要继续使用此订阅，无需进一步操作。如需更多功能，您可以升级到 Docker Pro 订阅。

要从 Docker Personal 升级到 Docker Pro，请参阅[升级您的订阅](./change.md#upgrade-your-subscription)。

## 为组织设置 Docker 订阅

您可以为新组织或现有组织订阅 Docker 订阅。只有组织所有者可以管理组织的账单。

在您[创建 Docker ID](../accounts/create-account.md) 后，您将拥有一个 Docker Personal 订阅。然后，您必须创建一个组织并为其选择订阅。有关更多详情，请参阅[创建您的组织](../admin/organization/orgs.md)。

要了解如何为现有组织升级 Docker 订阅，请参阅[升级您的订阅](./change.md#upgrade-your-subscription)。
