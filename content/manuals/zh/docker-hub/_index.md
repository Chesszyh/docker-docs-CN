Docker Hub 通过全球最大的容器注册表简化开发流程，用于存储、管理和共享 Docker 镜像。它与您的工具无缝集成，提高生产力并确保容器化应用程序的可靠部署、分发和访问。它还为开发人员提供预构建的镜像和资源，以加速开发工作流程。

Docker Hub 的主要功能：

* 无限的公共仓库
* 私有仓库
* 用于自动化工作流程的 Webhooks
* GitHub 和 Bitbucket 集成
* 并发和自动化构建
* 提供高质量、安全镜像的可信内容（Trusted Content）

除了图形界面外，您还可以使用 [Docker Hub API](../../reference/api/hub/latest.md) 或实验性的 [Docker Hub CLI 工具](https://github.com/docker/hub-tool#readme) 与 Docker Hub 进行交互。
